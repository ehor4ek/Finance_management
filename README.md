Консольное приложение для учета доходов и расходов.
Позволяет нескольким пользователям вести бюджет, устанавливать лимиты по категориям и получать отчеты о своих тратах.

Как пользоваться?

1. Установите Java 11 или новее и Maven 3.6 или новее
2. Скачайте проект или склонируйте репозиторий
3. Откройте терминал в папке проекта и выполните:

mvn clean compile
mvn exec:java

При первом запуске нужно зарегистрироваться:

1. Выберите "2. Регистрация"
2. Введите имя пользователя (минимум 3 символа)
3. Введите пароль (минимум 4 символа)
4. Подтвердите пароль

Основные команды

После входа вы увидите главное меню. Можно вводить команды двумя способами:

1. Выбирать цифры из меню (1, 2, 3...)
2. Писать команды напрямую

Основные команды:

Доходы и расходы:

add_income 5000 Зарплата Основная_зарплата
add_expense 300 Еда Продукты_на_неделю

Бюджеты:

set_budget Еда 5000
edit_budget Еда 6000
remove_budget Еда

Статистика:

balance                    # Показать баланс
stats 01.01.2024 31.01.2024 # Статистика за период
stats_month 01.2024        # Статистика за месяц
budgets                    # Показать все бюджеты
transactions              # Показать все транзакции
report 01.01.2024 31.01.2024 # Полный отчет

Категории:

add_category Развлечения
category_stats Еда,Развлечения # Статистика по категориям

Экспорт/импорт:

export my_data            # Экспорт в CSV
import my_data            # Импорт из CSV

Другие команды:

transfer Иван 1000 "На обед"  # Перевод другому пользователю
change_password              # Сменить пароль
alerts                      # Показать оповещения
logout                      # Выйти из аккаунта
exit                        # Выйти из приложения
help                        # Показать справку
clear                       # Очистить экран

Что умеет приложение?

1. Хранение данных - все данные хранятся в памяти, а при выходе сохраняются в файл
2. Несколько пользователей - каждый пользователь имеет свой логин и пароль
3. Категории - можно создавать свои категории для доходов и расходов
4. Бюджеты - можно устанавливать лимиты трат по категориям
5. Статистика - просмотр доходов и расходов за любой период
6. Оповещения - система предупреждает, если вы превысили бюджет или баланс низкий
7. Экспорт/импорт - можно сохранять данные в CSV файл и загружать из него
8. Переводы - можно переводить деньги другим пользователям

Оповещения

Приложение предупреждает о:

1. Превышении бюджета по категории
2. Когда потрачено больше 80% от лимита бюджета
3. Отрицательном балансе
4. Когда расходы превышают доходы
5. Низком балансе (менее 100 единиц)

Тестирование
В проекте есть тесты для проверки основных функций. Запуск тестов:

mvn test

Архитектура. Приложение разделено на слои (по директориям):

1. Модель - классы, которые хранят данные
2. Сервисы - содержат основную логику
3. Репозиторий - работает с хранением данных
4. CLI - обрабатывает команды пользователя
5. Исключения - обработка ошибок и исключительных ситуаций
